<clause id="sec-data-quality" obligation="normative">
<title>Data Quality</title>
<p id="_34b11636-2c0b-279b-bf0f-d898efe90ef6">Data quality allows users and user systems to assess fitness for use of the provided data. Data quality measures and the associated evaluation are reported as metadata of a data product. This metadata improves interoperability with other data products and provides usage by user groups that the data product was not originally intended for. The secondary users can make assessments of the data product usefulness in their application based on the reported data quality measures.</p>

<p id="_d55cc75e-dc77-c956-7e7f-f2f8c843ceea">The DCEG describes how data describing the real world should be captured using the types defined in the S-102 Feature Catalogue. The DCEG is located at <xref target="annex-data-classification-and-encoding-guide"/>.</p>
